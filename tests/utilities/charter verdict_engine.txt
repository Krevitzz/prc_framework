## SECTION 0 : RAPPEL FONDAMENTAL

### **Γ n'est pas l'objet étudié, Γ est l'instrument expérimental**

En R0, un mécanisme Γ est une **excitation différentielle** du système.
Il ne peut jamais être "fautif" par défaut.

**Échec d'un observable sous Γ peut invalider** :
- L'ordre tensoriel (ex : rang 2 insuffisant)
- La base de représentation (SYM vs ASY inadéquate)
- L'observable elle-même (test mal conditionné)
- Le mécanisme Γ (dans ce cadre spécifique)

**Conséquence** : Tout pattern détecté doit préciser **dans quelles conditions structurelles** il se manifeste.

## SECTION 2 : AXES D'ANALYSE (orthogonaux, non hiérarchisés)

### **Axe A : Γ (mécanisme)**
**Rôle** : Paramètre de variation, repère comparatif

**Questions** :
- Sous Γᵢ, observable X stable/explosive/trivial ?
- Γᵢ vs Γⱼ : qui révèle mieux phénomène Y ?
- Comportements généraux : quels Γ convergent, divergent, oscillent ?

**Formulations correctes** :
✅ "Sous GAM-001, SYM-001 conserve asymétrie"
✅ "GAM-013 amplifie tous observables (instabilité structurelle ?)"

**Formulations interdites** :
❌ "GAM-001 est bon"
❌ "GAM-013 échoue"

## SECTION 4 : DÉTECTION PATTERNS

### **4.1 Profiling gamma (prioritaire)**

**Pour chaque gamma** :
- Extraire runs de ce gamma uniquement
- Calculer comportement par test : stable/explosive/trivial
- Classifier régime : CONSERVES_X, DESTROYS_Y, NUMERIC_INSTABILITY

**Métriques par test** :
- `behavior` : stable, explosive, oscillating, trivial
- `regime` : classification qualitative (à définir par test)
- `final_value` : valeur finale médiane
- `trend` : slope normalisé

### **4.2 Comparaisons inter-gammas**

**Pour chaque test** :
- Classer gammas par conservation (meilleur → pire)
- Identifier outliers (≠ comportement majoritaire)
- Mesurer discriminabilité : variance entre gammas

### **4.3 Détection seuils (non systématique)**

**Approche** : Détection anomalies conditionnelles d'abord (B), approfondissement ciblé si nécessaire (A)

**Étape 1** : Scanner contextes à écart comportemental fort
```
Contexte (seed=4, M3) :
  - TOP-001 échoue 80% vs 5% global
  → Anomalie détectée
```

**Étape 2** : Si anomalie confirmée, exploration exhaustive triplets/quadruplets
```
Combinaisons testées :
  - (seed=4, M3, GAM-001) → échoue
  - (seed=4, M3, GAM-002) → stable
  → Affiner
```

**Critère lancement approfondissement** : Écart > 50% taux échec conditionnel vs marginal

## SECTION 5 : RÉGIMES COMPORTEMENTAUX

### **5.1 Taxonomie gamma**

**CONSERVES_SYMMETRY** : asymétrie finale < 1e-6
**CONSERVES_NORM** : norme finale < 2× initiale
**CONSERVES_PATTERNS** : diversity/uniformity stable
**NUMERIC_INSTABILITY** : ≥1 observable > 1e50
**DESTROYS_TOPOLOGY** : euler_characteristic change signe
**TRIVIAL** : aucune variation observable (σ/μ < 0.01)

### **5.2 Taxonomie tests**

**DISCRIMINANT** : variance entre gammas > 0.5
**NON_DISCRIMINANT** : variance < 0.1
**NUMERICALLY_UNSTABLE** : >10% runs explosifs
**WELL_CONDITIONED** : <1% explosions, variance exploitable

### **PHASE 2 : CORRECTIONS CRITIQUES (biais structurels)**

#### **2.1 Variance marginale (PRIORITÉ 1)**

**Problème actuel** :
```python
group_means = [np.mean(g) for g in factor_groups]
var_between = np.var(group_means)  # ← variance de moyennes
var_total = np.var(group[projection].dropna())  # ← variance brute
variance_ratio = var_between / var_total  # ← INCOMPATIBLE
```

**Correction (ANOVA-like)** :
```python
def compute_variance_ratio_corrected(group, factor, projection):
    """
    Variance ratio corrigé : SSB / (SSB + SSW)
    
    SSB = variance entre groupes (between)
    SSW = variance intra-groupes (within)
    """
    groups = [g[projection].dropna().values for _, g in group.groupby(factor)]
    
    # Grand mean
    grand_mean = np.mean(np.concatenate(groups))
    
    # SSB (Sum of Squares Between)
    ssb = sum(len(g) * (np.mean(g) - grand_mean)**2 for g in groups)
    
    # SSW (Sum of Squares Within)
    ssw = sum(np.sum((g - np.mean(g))**2) for g in groups)
    
    # Variance ratio (η²-like)
    variance_ratio = ssb / (ssb + ssw) if (ssb + ssw) > 1e-10 else 0.0
    
    return variance_ratio, ssb, ssw
```
