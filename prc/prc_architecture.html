01 — Execution Flow
batch.py
Point d'entrée CLI. 3 modes : run+verdict / --verdict {phase} / --verdict (cross-phases)
args: phase, --regime-profile configs/phases/poc/{phase}.yaml
hub_running.run_batch()
Génère compositions (produit cartésien axes YAML) → benchmark 1 sample/rank → dry-run stats → confirmation o/n → boucle runs
yaml_path, auto_confirm rows: List[Dict], parquet_path
runner.run_single()
prepare_state() → run_kernel() → history np.ndarray (T, *dims). Détection explosion NaN/Inf → break propre.
composition dict history: ndarray (T, *dims)
hub_featuring.extract_features()
inspect_history() → métadonnées → boucle layers YAML → check_applicability() → extract_for_layer()
history, features_config (load_all_configs) {features: dict, layers: list} ⚠ universal jamais appliqué (bug applicability)
write_parquet()
Écriture Parquet avant verdict (resilience). 1 fichier par phase.
rows (composition + features + layers) data/results/{phase}.parquet
verdict.run_verdict_intra()
Opère sur rows en mémoire. Séquence : run_profiling() → run_analysing() → analyze_outliers() → classify_regimes_batch()
rows (RAM) ❌ outliers: n_features_used=0 (intersection vide) ❌ régimes: 2/8 (norm_initial=1.0 défaut)
write_verdict_report()
JSON + TXT lisible humain.
reports/verdict_{phase}.json reports/verdict_{phase}.txt

06 — Known Issues (phase ironing)
BUG-01 — check_applicability : opérateurs non gérés
featuring/layers_lite.py → check_applicability()
Gère uniquement les comparaisons Python natives (bool, int, float, list). Les strings spéciales "any" et ">=3" tombent dans le else et retournent False systématiquement.
→ universal.yaml : rank: any → universal jamais extrait → pas de euclidean_norm_initial/final
→ tensor_3d.yaml : rank: ">=3" → tensor_3d jamais extrait
→ cascade : intersection features vide → outliers n_features_used=0 + norm_initial=1.0 → régimes 2/8
BUG-02 (cascade) — Régimes 2/8 détectés
analysing/regimes_lite.py → classify_regime()
norm_initial = features.get('euclidean_norm_initial', 1.0) — défaut 1.0 car universal non extrait. Ratio = norm_final / 1.0. Majorité runs : norm_final < 1.3 → tout CONSERVES_NORM. EFFONDREMENT/CROISSANCE/SATURATION non atteignables.
Résolu par correction BUG-01.
BUG-03 (cascade) — outliers n_features_used=0
analysing/outliers_lite.py → _extract_common_features()
Intersection features matrix_2d ∩ tensor_3d = ∅ (aucune feature commune). universal absent → pas de euclidean_norm qui serait la feature commune naturelle. IsolationForest non appelé.
Résolu par correction BUG-01.
